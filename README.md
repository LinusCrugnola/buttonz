## Features

Custom buttons

## Requirements

None

## Extension Settings

You can define the commands that are run by the buttons in settings.json:
```json
{
    "buttonz": {
        "launch": "echo 'Button 1'",
        "build" : "echo 'Button 2'"
    }
}
```
If nothing is specified, a default echo command is executed.
